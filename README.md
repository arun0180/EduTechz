# Light Of Education  
A simple, elegant, and interactive educational website designed to help students access important academic content easily.  
This project supports **Kannada** and **English**, with subject-wise navigation for grammar, math formulas, tourist places, famous personalities, quiz sections, and more.

# Features

#Multi-Language Support
- Instant switch between **English** and **Kannada**
- Dynamic text update without reloading the page

#Clean & Responsive UI
- Fully responsive for mobile and desktop  
- Smooth color themes for each subject  
- Semi-transparent background overlay for readability  

#Subject Sections
- Kannada Grammar  
- English Grammar  
- Hindi Grammar  
- Math Formulas  
- Tourist Places  
- Famous Personalities  
- Quiz & Puzzles  
- Kannada Jnanapeeta Award Winners  

#Tech Stack
- **HTML5**  
- **CSS3**  
- **JavaScript** (inline, no external JS file needed)  
- **GitHub Pages** for hosting
